# 🚀 医院排班系统 - Windows 版使用说明

## 📋 前提条件

在使用一键脚本之前，请确保你的电脑已安装：

1. **Node.js**（必装）
   - 下载地址：https://nodejs.org/
   - 选择 LTS 版本（推荐 18.x 或 20.x）
   - 安装完成后重启电脑

2. **Git**（必装）
   - 下载地址：https://git-scm.com/download/win
   - 安装时选择默认选项即可

---

## 🎯 快速开始（3 步搞定）

### 步骤 1：获取代码

**方式 A：如果你已有代码**
- 直接打开项目文件夹

**方式 B：从 GitHub 克隆**
```bash
git clone https://github.com/orange1168/hospital-schedule.git
cd hospital-schedule
```

### 步骤 2：一键打包

1. 在项目文件夹中找到 **`一键打包.bat`**
2. 右键 → "以管理员身份运行"
3. 等待自动完成（首次约 5-10 分钟）

**脚本会自动完成：**
- ✓ 检查和安装必要工具
- ✓ 设置国内镜像（加速下载）
- ✓ 安装所有依赖
- ✓ 修改配置文件
- ✓ 编译 H5 版本
- ✓ 打包 Windows 应用

### 步骤 3：一键启动

1. 找到 **`一键启动.bat`**
2. 双击运行
3. 自动启动后端和桌面应用

## 🐛 常见问题

### 问题 1：双击脚本没反应

**解决**：
1. 右键 → "以管理员身份运行"
2. 检查是否已安装 Node.js 和 Git

### 问题 2：提示 "node: command not found"

**解决**：
1. 下载安装 Node.js：https://nodejs.org/
2. 安装后重启电脑
3. 重新运行脚本

### 问题 3：提示 "git: command not found"

**解决**：
1. 下载安装 Git：https://git-scm.com/download/win
2. 安装后重启电脑
3. 重新运行脚本

### 问题 4：打包失败，提示下载超时

**解决**：
- 脚本已自动设置国内镜像
- 如果仍然失败，请检查网络连接
- 尝试重新运行脚本

### 问题 5：打包成功，但应用无法启动

**解决**：
1. 确保 **先启动后端**：运行 `一键启动.bat` 或手动运行 `pnpm dev:server`
2. 检查 3000 端口是否被占用
3. 查看应用窗口的错误提示

### 问题 6：应用无法连接后端

**解决**：
1. 检查后端是否正在运行
2. 在浏览器访问 http://localhost:3000 检查后端状态
3. 查看后端窗口的错误日志

## 💡 使用技巧

### 技巧 1：手动启动（更灵活）

如果你需要分开启动后端和前端：

**启动后端**：
```bash
pnpm dev:server
```

**启动应用**：
- 双击 `dist\electron\win-unpacked\医院排班系统.exe`

### 技巧 2：创建桌面快捷方式

1. 右键 `dist\electron\win-unpacked\医院排班系统.exe`
2. 选择"发送到" → "桌面快捷方式"
3. 以后可以直接从桌面启动应用

### 技巧 3：修改应用名称

如果想修改应用名称，编辑 `package.json`：
```json
{
  "name": "hospital-schedule",
  "productName": "你的应用名称"
}
```
然后重新打包。
